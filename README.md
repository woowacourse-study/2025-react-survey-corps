# 📚 커밋 & 브랜치 컨벤션

### 브랜치 규칙

```bash
week${주차}
```

예시: week1, week5

### 커밋 메시지

```bash
{n}주차: {커밋 메시지}
```

예시: 3주차: 훅 개념 정리 및 예제 추가
